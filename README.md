# Video Encryptor

![Video Encryptor](https://github.com/MrMDrX/Video-Encryptor/blob/main/images/video_encryptor_banner.png)

Video Encryptor is a Python application that allows users to compress, encrypt, and decrypt video files. It provides a simple and intuitive graphical interface for selecting files and applying encryption algorithms. The application is designed to enhance data security and facilitate the secure storage and transfer of video content.

## Features

- Select video files for compression, encryption, and decryption.
- Compress video files using popular video codecs.
- Encrypt video files using a secret key for secure storage and transfer.
- Decrypt encrypted video files using the same secret key.
- Dark mode and Light mode
- Intuitive graphical interface

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/MrMDrX/Video-Encryptor.git
   ```

2. Navigate to the Project Directory:

   ```bash
   cd Video-Encryptor
   ```

3. Create and Activate a Virtual Environment (Optional but recommended):

   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows, use 'venv\Scripts\activate'
   ```

4. Install the required dependencies:

   ```sh
   pip install -r requirements.txt
   ```

5. Run the application:

   ```sh
   python main.py
   ```

## Usage

1. Launch the application.
2. Select the desired operation (Encrypt, Decrypt, etc.) from the navigation menu.
3. Follow the on-screen instructions to select the input file and perform the chosen operation.
